# Data spec

## Dates Blacklist
Dates can be black listed to not be used when generating times.

File: _date-blacklist.csv_

Format:

MONTH-DAY

Description:

MONTH - Month expressed as integer:
        01 - January;
        02 - February;
        03 - March;
        04 - April;
        05 - May;
        06 - June;
        07 - July;
        08 - August;
        09 - September;
        10 - October;
        11 - November;
        12 - December;
        
DAY - the day prepended with 0 if needed to have two digits :)
